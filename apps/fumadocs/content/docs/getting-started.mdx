## Prerequisites

- Bun 1.2+
- PostgreSQL (if running server)

## Install

```bash
bun install
```

## Environment

```bash
cp apps/server/.env.example apps/server/.env
```

## Database (optional)

```bash
bun db:push
```

## Run

- All apps:
```bash
bun dev
```
- Web only:
```bash
bun dev:web
```
- Server only:
```bash
bun dev:server
```

Web: http://localhost:3001
API: http://localhost:3000

## Data Source

The storefront reads demo data from `data/data.ts`.
