title: Overview
description: What is mc-webstore
---

mc-webstore is a Turborepo monorepo featuring:

- apps/web: Next.js storefront using TailwindCSS and shadcn/ui
- apps/server: Next.js API with oRPC, Better Auth, Drizzle ORM, PostgreSQL
- apps/fumadocs: Fumadocs-powered docs site
- data/data.ts: demo catalog data consumed by the storefront

Goals:

- Type-safe end-to-end development
- Modular architecture with clear app boundaries
- Fast local dev using Bun + Turbo
